== Description ==

By using this plugin you can check your website page speed with Google Insight API [https://developers.google.com/speed/pagespeed/insights/](Google Insights API). You can also increase website performance by 
given script like Enable GZIP Compression, Leverage Browser Caching and defer javascript.

Use plugin script in your .htaccess file and function.php file to increase your website performance.

== Installation ==

1. Upload `website-speed-checker` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter Website URL and select strategy like desktop/mobile get all website performance report.
4. Use performance increase script to increase performance.
